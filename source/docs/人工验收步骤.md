## 附录 B：常见问题

**Q: 登录后页面空白或 API 返回 500？**
A: 后端 JWT 未认证的请求日志显示为 401/500，这是 NestJS Passport 的已知行为。登录后前端通过 axios 拦截器自动携带 Token，不影响正常使用。

**Q: 数据库有测试数据如何清理？**
A: 运行种子脚本重建数据：`cd source/backend && npm run seed`。这会重建管理员账号并清空其他数据。

**Q: 文件上传路径在哪？**
A: 上传的文件存储在 `source/backend/uploads/orders/{orderId}/` 目录下。
